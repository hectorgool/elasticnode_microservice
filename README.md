# ElasticSearch.js Microservice

### 1. Create a proyect:
```sh
npm init
```
the line above, creates the package.json file.
### 2. Install express:
```sh
npm install express --save
```
add dependencies to the package json file.
### 3. Create git repotitory:
```sh
git init
git add .
git commit -m “install”
```
